Zmienna to nie wartość

LET vs CONST

CONST to stałe połączenie z jedną wartością i tej wartości nie możemy zmienić

LET to zmienne połączenie które poprostu możemy zmienić.

Zmienne w JS są jak młody pelikan - łykną wszystko.

Typy zmiennych:
String - opakowujemy w apostrofy ' '
Number - liczba
BigInt - liczba większa niż 2 do potęgi 53
Boolean - true lub false
null - wrtość celowo niezdefiniowana którą później zmieniamy
undefined - wartość która nie jest zdefiniowana
*symbol - tworzy unikalne opakowanie

Sprawdzenie jakiego typu jest zmienna - TYPEOF
