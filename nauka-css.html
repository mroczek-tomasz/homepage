Nauka CSS

Co tam panie w CSS?

Czego się nauczyłem?

Co to jest visual code studio i trochę jego interfejsu.

Własność: display

block - element zajmuje całą szerokość strony i elementy ustawiają się pod sobą.
inline - element zmniejsza się do szerokości jaką zajmuje i wyświetlane są koło siebie.
inline-block: dodawanie własności blokowych, na przykład szerokość i wysokość, do właściwości liniowych.

Własność: min i max do własności wysokości i szerokości

Własność: overflow - odcinanie tego co nam wystaje

hidden - chowa wartość która jest poza divem.
visible - domyślna wartość czyli wystaje poza diva
auto - często dodaje nam scrola wewnątrz elementu.

Jednostki

px - pixele

% - procenty (procenty względem rodzica) - raczej szerokość a nie wysokość

vw - viewport width - szerokość całego okna

vh - viewport height - wysokość całego okna
