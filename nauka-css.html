Nauka CSS

Czego się nauczyłem?

Własność: display

block - element zajmuje całą szerokość strony i elementy ustawiają się pod sobą.
inline - element zmniejsza się do szerokości jaką zajmuje i wyświetlane są koło siebie.
inline-block: dodawanie własności blokowych, na przykład szerokość i wysokość, do właściwości liniowych.

Własność: min i max do własności wysokości i szerokości

Własność: overflow - odcinanie tego co nam wystaje

hidden - chowa wartość która jest poza divem.
visible - domyślna wartość czyli wystaje poza diva
auto - często dodaje nam scrola wewnątrz elementu.

Jednostki

px - pixele

% - procenty (procenty względem rodzica) - raczej szerokość a nie wysokość

vw - viewport width - szerokość całego okna

vh - viewport height - wysokość całego okna

Box model

wartości są clockwise - top right bottom left

paddding - odległość wewnętrzna do ramki

border czyli ramka - określasz: szerokość, styl, kolor.
border-radius - zaokrąglenie rogów.

margin - margines czyli obszar poza ramką.
margin:auto - wyśrodkowuje element.

Box sizing

content-box - wartość domyślna - pudełko ograniczone contentem

border-box - pudełko ograniczone borderem.
Określasz jaką wartość ma mieć content razem z borderem, a reszta wylicza się sama.

Position

static - domyślna pozycja.

relative - relatywna

absolute -
top : 20px
right, left, bottom

z-index - dodaje tak jakby trzeci wymiar - im większa liczba tym obiekt jest bliżej widza.

fixed - jest zafixowany na stronie - wartości top right left bottom

sticky - jest przyklejony góry strony jeżeli uderzamy w niego z góry.

Float - nie służy do budowania layoutów

left - tak jakby display był inline, a tekst opływał dany element.

right - to samo tylko na prawo

clear - czyści to co wystaje poza element

overflow: hidden -też czyści to wystaje.
